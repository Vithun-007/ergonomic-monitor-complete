# Smart Ergonomic Risk Monitor

This repository contains the complete pipeline for a real-time ergonomic monitoring system.

## Steps
1. step1_camera – Webcam verification
2. step2_pose – Pose estimation using MediaPipe
3. step3_inverse_kinematics – Joint angle computation (to be implemented)
4. step4_ergonomic_scoring – RULA/REBA scoring (to be implemented)
5. step5_temporal_risk – Temporal exposure modeling
6. step6_injury_prediction – RSI prediction model
7. step7_dashboard – Visualization dashboard

Open this folder once in VS Code and work step-by-step.
